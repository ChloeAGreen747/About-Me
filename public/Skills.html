What is CSS and ECMAScript?

CSS stands for Cascading Style Sheet. It is used to show how HTML elements are to be displayed on screen by referencing the individual elements that make up the HTML page. CSS controls the background colour, text fonts and colours and controls where each element is to be placed on the page.
