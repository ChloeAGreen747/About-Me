What is Software Development?

Software is a set of instructions or programs that tell a computer what to do. Software development is the process of creating, designing, deploying and supporting software. To be a Software Developer you should be able to analyze users needs in order to develop the software that meets those needs.

Why Did I Choose Software Development?

I chose Software Development as my favourite part of Computer Science is programming. I also love maths and problem solving which is a key part of being able to program. When I finish university I hope to become a software developer as it combines the two things I enjoy.

What Skills Do You Need For Software Development?

  • Teamwork and good communication skills are required since most software is developed by a team of developers
  • Be able to use different programming languages
  • You should have good written skills so that other people can understand and read your code.
  • You need to be good at problem solving not only to create the software but to find errors in the code more efficiently
